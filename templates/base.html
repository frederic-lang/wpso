Bienvenue sur WPSO, l'assistant de preuve web dans l'arithmétique de Peano. Ici, vous pouvez démontrer formellement des théorèmes mathématiques, c'est-à-dire en construire avec le calcul des séquents dans la section demonstrate, puis l'enregistrer. Il est possible d'utiliser dans une preuve les lemmes déjà prouvés.

Allez à la section Library pour voir toutes les propositions établies. La section Doc regroupe la documentation des instructions qui vous seront nécessaires pour ensuite établir votre théorème, dans la section Demonstrate. Enfin, la section Community est la partie vivante du site, pour échanger et suivre les avancée tant dans les lemmes démontrés que dans le developpement informatique du projet.

!! Le site est toujours en phase de développement, il est normal de tomber sur des disfonctionnements, n'hésitez pas à les signalez sur Github. En particuler, il manque la multiplication. Rejoignez aussi la page Github pour plus d'informations et poser des questions.
